Página pessoal no estilo Linktree para reunir links importantes em um único lugar. O projeto foi feito com HTML e CSS, com foco em simplicidade, apresentação pessoal e organização visual.

## Conteúdo

- Página `index.html`.
- Arquivo de estilos `style.css`.
- Imagem de perfil.
- Links organizados em uma página única.

## Tecnologias

- HTML5
- CSS3
- Git e GitHub

## Como visualizar

Acesse pelo GitHub Pages:

https://matheusgdqueiroz-del.github.io/Linktree/

Ou baixe o repositório e abra o arquivo `index.html` no navegador.
