# Linktree

Pagina pessoal no estilo Linktree para reunir links importantes em um unico lugar. O projeto foi feito com HTML e CSS, com foco em simplicidade, apresentacao pessoal e organizacao visual.

## Conteudo

- Pagina `index.html`.
- Arquivo de estilos `style.css`.
- Imagem de perfil.
- Links organizados em uma pagina unica.

## Tecnologias

- HTML5
- CSS3
- Git e GitHub

## Como visualizar

Acesse pelo GitHub Pages:

https://matheusgdqueiroz-del.github.io/Linktree/

Ou baixe o repositorio e abra o arquivo `index.html` no navegador.
